# Cybrium AI — Homebrew Tap

Homebrew tap for Cybrium AI security tools.

## Install

```bash
brew tap cybrium-ai/tap
brew install cyweb        # Cybrium Web — fast web vulnerability scanner
# brew install cyscan     # (coming)
# brew install cyprobe    # (coming)
```

## Available formulas

| Formula | Description | Source |
|---|---|---|
| `cyweb` | Web vulnerability scanner (Nikto-class) | [cybrium-ai/cyweb](https://github.com/cybrium-ai/cyweb) |

Formulas in this repo are auto-published by each tool's release pipeline.
